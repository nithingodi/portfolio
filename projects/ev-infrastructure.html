EV Infrastructure Planning

Mapped 200K+ EV records and built dashboards for underserved Washington counties.

← Back to Projects

Project Overview

Developed geospatial dashboards on 200K+ EV registrations and 2.5K charging stations to target underserved Washington counties.

The Challenge

Washington needed to allocate limited IIJA funding to counties where EV adoption outstripped charging access.

Solution Approach

  • Curated six datasets (EV registrations, station locations, population density, traffic flows, road networks, jurisdictions) in SQL and Python.
  • Built interactive Tableau dashboards to score each county on vehicles-per-station and demand indicators.

Results & Impact

  • Flagged 5 priority counties (Snohomish, Kittitas, Skagit, Cowlitz, Lewis) with demand-to-capacity ratios >2.1×.
  • Recommendations adopted into the state’s 2024 charging-network rollout plan.

Project Details

Tools Used: Tableau, Geospatial Analysis, Dashboard Design, SQL, Python

Key Impact: Identified 5 priority counties for EV infrastructure
