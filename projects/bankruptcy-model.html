Bankruptcy Prediction Model

Built and compared 8 ML models; best ensemble had lowest false negatives.

← Back to Projects

Project Overview

Benchmarked 8 algorithms in SAS EM and deployed a five-node ensemble to minimize false negatives in corporate-failure forecasts.

The Challenge

False negatives carry outsized financial risk—model must err on the side of flagging potential bankruptcies.

Solution Approach

  • Preprocessed financial ratios and categorical factors.
  • Experimented with Polynomial Logistic, forward/backward selection, Lasso, Decision Tree, Gradient Boosting, Neural Network; tuned via 10-fold cross-validation.

Results & Impact

Ensemble model achieved the lowest false-negative rate (3.2%) on hold-out data, outperforming standalone methods by 15–23%.

Project Details

Tools Used: SAS EM, Machine Learning, Ensemble Methods

Key Impact: Reduced false negatives by 35%
